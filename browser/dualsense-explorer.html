DualSense Explorer

Use the WebHID API to connect to a Sony PlayStation 5 DualSense Wireless Controller.

Instructions

Connect the controller to your computer using either Bluetooth or a USB cable.

Then, click , select Wireless Controller and click the Connect button.

This page requires a Chromium-based browser like Google Chrome or Microsoft Edge. It uses WebHID API, which is only available on Windows, macOS, chromeOS, and Linux.
